## Effective Temperature T_eff

The effective temperature is computed from gradient covariance at the final training epoch:

```math
T_{\text{eff}} = \frac{1}{d} \text{Tr}(\Sigma)
```

where:
- **Σ**: Gradient covariance matrix at convergence
- **d**: Number of parameters (21 for Strassen 7-slot model)
- **Tr**: Matrix trace

### Computing T_eff from Gradients

```python
def compute_effective_temperature(model, dataloader, num_batches=50):
    """Compute T_eff from gradient covariance"""
    model.eval()
    gradients = []
    
    for i, (A, B, C) in enumerate(dataloader):
        if i >= num_batches:
            break
        
        C_pred = model(A, B)
        loss = nn.functional.mse_loss(C_pred, C)
        
        # Compute gradients
        grad = torch.autograd.grad(
            loss, 
            model.parameters(), 
            create_graph=False
        )
        
        # Flatten all parameter gradients
        grad_flat = torch.cat([g.flatten() for g in grad])
        gradients.append(grad_flat)
    
    # Stack into matrix [num_batches × num_params]
    gradients = torch.stack(gradients)
    
    # Compute covariance matrix
    Sigma = torch.cov(gradients.T)
    
    # Effective temperature = trace / d
    d = Sigma.shape[0]
    T_eff = torch.trace(Sigma).item() / d
    
    return T_eff, Sigma
```

## Gradient Covariance Condition Number κ

The condition number of the gradient covariance matrix acts as an **order parameter**:

```math
\kappa = \frac{\lambda_{\max}(\Sigma)}{\lambda_{\min}(\Sigma)}
```

### κ as a Phase Indicator

```python
def predict_crystallization(model, dataloader, epoch):
    """Predict if model will crystallize based on current κ"""
    _, Sigma = compute_effective_temperature(model, dataloader)
    
    # Add regularization for numerical stability
    d = Sigma.shape[0]
    Sigma_reg = Sigma + 1e-8 * torch.eye(d, device=Sigma.device)
    
    # Compute condition number
    eigenvalues = torch.linalg.eigvalsh(Sigma_reg)
    eigenvalues = eigenvalues[eigenvalues > 1e-10]
    
    if len(eigenvalues) == 0:
        kappa = float('inf')
    else:
        kappa = (eigenvalues.max() / eigenvalues.min()).item()

    will_crystallize = (kappa < 1.1)
    
    print(f"Epoch {epoch}: κ = {kappa:.2f}")
    print(f"Prediction: {'CRYSTAL' if will_crystallize else 'GLASS'}")
    
    return will_crystallize, kappa
```

## Discretization Margin δ

The discretization margin measures how close weights are to integer values:

```math
\delta(\theta) = \|\theta - Q(\theta)\|_\infty
```

where Q(θ) is the discretization operator (round and clamp to {-1, 0, 1}).

```python
def compute_delta(model):
    """Compute discretization margin"""
    coeffs = model.get_coefficients()
    
    deltas = []
    for name, tensor in coeffs.items():
        # Distance to nearest integer
        delta = (tensor - tensor.round()).abs().max().item()
        deltas.append(delta)
    
    return max(deltas)
```

## Entropy and Heat Capacity

### Entropy from Microstate Distribution

Treating final weights as microstates gives a partition function and entropy:

```python
def compute_entropy(checkpoints):
    """Compute entropy from checkpoint distribution"""
    all_params = []
    
    for ckpt in checkpoints:
        coeffs = ckpt['model'].get_coefficients()
        params = torch.cat([c.flatten() for c in coeffs.values()])
        all_params.append(params.cpu().numpy())
    
    all_params = np.stack(all_params)
    
    # Covariance-based entropy
    cov = np.cov(all_params.T)
    cov += 1e-8 * np.eye(cov.shape[0])  # Regularization
    
    eigenvals = np.linalg.eigvalsh(cov)
    eigenvals = eigenvals[eigenvals > 1e-10]
    
    if len(eigenvals) == 0:
        return 0.0
    
    # Differential entropy (Gaussian approximation)
    log_prod = np.sum(np.log(eigenvals))
    S = 0.5 * log_prod + len(eigenvals) * 0.5 * np.log(2 * np.pi * np.e)
    
    return float(S)
```

### Heat Capacity C_v

The heat capacity measures how entropy changes with temperature:

```math
C_v = \frac{\partial S}{\partial T} \bigg|_V
```

Near the phase transition, heat capacity diverges—a signature of first-order transitions.

The Second Law manifests as entropy production during generalization:

```python
def analyze_entropy_production(model, trajectory_length=100):
    """Track entropy evolution during training"""
    
    # Simulate training trajectory
    initial_params = torch.randn_like(
        torch.cat([p.flatten() for p in model.parameters()])
    ) * 0.1
    
    final_params = torch.cat(
        [p.flatten() for p in model.parameters()]
    ).detach()
    
    entropy_values = []
    
    for t in np.linspace(0, 1, trajectory_length):
        # Interpolate between initial and final states
        params = initial_params * np.exp(-t * 3) + \
                 final_params * (1 - np.exp(-t * 3))
        
        # Add annealing noise
        noise_scale = 0.1 * (1 - t)
        params += torch.randn_like(params) * noise_scale
        
        # Compute generalization entropy
        S_gen = compute_generalization_entropy(params)
        entropy_values.append(S_gen)
    
    # Entropy production rate
    dS_dt = np.gradient(entropy_values)
    
    return {
        'initial_entropy': entropy_values[0],
        'final_entropy': entropy_values[-1],
        'entropy_drop': entropy_values[0] - entropy_values[-1],
        'max_production_rate': np.max(np.abs(dS_dt))
    }
```

## Quantum-Regularized Metrics

For states with near-zero variance (perfect crystals), classical covariance becomes singular. Quantum regularization resolves this:

### κ_quantum

```python
def compute_kappa_quantum(coeffs, hbar=1e-6):
    """Quantum-regularized condition number"""
    flat_params = torch.cat([c.flatten() for c in coeffs.values()])
    n = flat_params.numel()
    
    if n < 2:
        return 1.0  # Perfect ground state
    
    # Manual covariance with quantum regularization
    params_centered = flat_params - flat_params.mean()
    cov_matrix = torch.outer(params_centered, params_centered) / n
    cov_matrix = cov_matrix + hbar * torch.eye(n, device=flat_params.device)
    
    eigenvals = torch.linalg.eigvalsh(cov_matrix)
    eigenvals = eigenvals[eigenvals > hbar]
    
    if len(eigenvals) == 0:
        return 1.0
    
    return (eigenvals.max() / eigenvals.min()).item()
```

### α Purity

The α purity metric is an inverse temperature for discrete states:

```math
\alpha = -\log(\delta)
```

- **α > 7**: Crystal states
- **α < 3**: Glass states
- **3 ≤ α ≤ 7**: Polycrystalline (intermediate)

```python
def compute_alpha_purity(coeffs):
    """Alpha purity: inverse temperature metric"""
    delta = compute_delta(coeffs)
    
    if delta < 1e-10:
        return 20.0  # α_max for perfect discretization
    
    return -np.log(delta)
```

## Effective Planck Constant ℏ_eff

The system exhibits a minimum "quantum" of gradient noise:

```math
\hbar_{\text{eff}} = \Delta\theta \cdot \kappa_q
```

where Δθ is parameter uncertainty in the symmetry basis.

<Info>
**Physical Interpretation**: There is a minimum effective temperature for algorithm crystallization. Batch sizes control ℏ_eff:

- Small batches (B < 24): ℏ_eff too high, unstable
- Large batches (B > 128): ℏ_eff increases entropy (h_bar), traps in glass
- Optimal (B ∈ [24, 128]): ℏ_eff in crystallization range
</Info>
