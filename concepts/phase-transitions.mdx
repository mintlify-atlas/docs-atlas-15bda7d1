## Overview

Neural network training exhibits **first-order phase transitions** analogous to materials science: networks transition abruptly from high-entropy amorphous glass states to low-entropy crystalline algorithmic structures. This transition is characterized by discontinuous changes in order parameters with no intermediate states.

<Note>
**Key Evidence**: In 60 hyperparameter configurations, every successful run shows κ = 1.000, while every failed run shows κ = 999999. The confidence interval appears degenerate because **there are no intermediate points**—this is a first-order phase transition.
</Note>

## The Order Parameter Φ

The order parameter for algorithmic crystallization is defined as:

```math
\Phi = \mathbb{1}[\delta = 0]
```

where Φ = 1 indicates successful discretization (crystal) and Φ = 0 indicates failure (glass).

### 2. Polycrystalline Phase (3 ≤ α ≤ 7)

An intermediate state with partial crystalline structure:

- **Mixed structure**: Some slots discretized, others continuous
- **Generalizes** on test sets but fails full verification
- **Created by** over-pruning (50-70% sparsity)
- **Unstable**: Small perturbations cause collapse

<Warning>
Polycrystalline states arise when pruning exceeds the 50% threshold. The solution retains generalization capability but loses exact algorithmic structure.
</Warning>

### 3. Amorphous Glass Phase (α < 3)

- **Continuous weights**: Far from any integer values
- **Local minimum**: Generalizes on test set but non-algorithmic
- **High curvature**: Anisotropic gradient covariance (κ → ∞)
- **High entropy**: Large h_bar, dense parameter distribution

```python
def classify_phase(model):
    """Classify material phase"""
    coeffs = model.get_coefficients()
    
    delta = compute_delta(coeffs)
    alpha = compute_alpha_purity(coeffs)
    kappa = compute_kappa_quantum(coeffs)
    
    if alpha > 7 and delta < 0.01:
        return "Crystalline", {
            'grade': 'Optical',
            'color': 'green',
            'stability': 'Excellent'
        }
    elif alpha > 7 and delta < 0.1:
        return "Crystalline", {
            'grade': 'Industrial',
            'color': 'blue',
            'stability': 'Good'
        }
    elif 3 <= alpha <= 7:
        return "Polycrystalline", {
            'grade': 'Mixed',
            'color': 'yellow',
            'stability': 'Moderate'
        }
    else:
        return "Glass", {
            'grade': 'Amorphous',
            'color': 'red',
            'stability': 'Poor'
        }
```

## The Grokking Transition

Grokking is the observable manifestation of the phase transition:

<img src="/images/grokking-transition.png" alt="Grokking phase transition" />

### Three Stages

<Steps>
  <Step title="Pre-Grok (Epochs 0-1000)">
    **High-entropy exploration**
    
    - Training loss drops rapidly
    - Test loss remains high (>0.1)
    - LC ≈ 0.95 (maximum complexity)
    - Weights distributed broadly
  </Step>
  
  <Step title="Grokking Event (Epoch ~2160)">
    **Phase transition**
    
    - Test loss drops abruptly (greater than 0.1 to less than 1e-6)
    - LC falls from 442 → 0
    - κ transitions: ∞ → 1.000
    - δ drops: 0.49 → 0
  </Step>
  
  <Step title="Post-Grok (Epochs 2160+)">
    **Crystalline consolidation**
    
    - Both losses < 10⁻⁶
    - Weights converge to integers
    - T_eff drops to 10⁻¹⁷
    - System enters discrete basin
  </Step>
</Steps>

### Local Complexity as Grokking Detector

Local Complexity (LC) captures the phase transition:

```python
def compute_local_complexity(model):
    """LC metric from Can't Stop Won't Stop paper"""
    params = torch.cat([p.flatten() for p in model.parameters()])
    
    # Use percentiles for robustness
    perc_95 = torch.quantile(torch.abs(params), 0.95)
    active = (torch.abs(params) > 0.01 * perc_95).sum()
    lc = active.float() / len(params)
    
    return lc.item()
```

**Key observation**: LC falls to exactly zero at the grokking transition (epoch 2160), confirming that LC captures the phase change.

## Topological Surgery Interpretation

The phase transition is **topological surgery** on the weight manifold:

### Pre-Grokking: High-Curvature Glass

```math
\mathcal{M}_{\text{glass}} = \{\theta : \kappa(\Sigma(\theta)) \gg 1\}
```

The weight manifold has:
- High curvature
- Anisotropic geometry
- Multiple local minima
- Dense parameter distribution

### Post-Grokking: Hypersphere Crystal

```math
\mathcal{M}_{\text{crystal}} = \{\theta : \kappa(\Sigma(\theta)) = 1\}
```

The manifold undergoes surgery to become:
- **Perfectly spherical** (uniform curvature)
- **Topologically simple** (Perelman's hypersphere)
- **Minimum entropy** (unique discrete attractor)
- **Zero effective dimension** (all weights integer)

<Note>
**Connection to Poincaré Conjecture**: Perelman's proof shows that Ricci flow with surgery transforms any simply connected 3-manifold into a sphere. The discrete Strassen solution is the **topologically simplest form** of the matrix multiplication manifold—the sphere is the unique zero-entropy attractor.
</Note>

## Entropy Scaling Law

Entropy production follows an exponential decay:

```math
S(t) = S_0 e^{-t/\tau} + S_\infty
```

where:
- **S₀**: Initial entropy (high)
- **S_∞**: Final entropy (→0 for crystal)
- **τ**: Timescale (varies with α)

```python
def fit_entropy_scaling(entropy_trajectory):
    """Fit exponential decay to entropy evolution"""
    def model(t, S0, tau, Sinf):
        return S0 * np.exp(-t / tau) + Sinf
    
    t = np.arange(len(entropy_trajectory))
    
    popt, _ = curve_fit(
        model, 
        t, 
        entropy_trajectory,
        p0=[entropy_trajectory[0], 50, entropy_trajectory[-1]]
    )
    
    return {
        'S0': popt[0],
        'tau': popt[1],
        'Sinf': popt[2]
    }
```
