## Overview

The two-phase protocol is the core engineering methodology for inducing algorithmic structure in neural networks. It consists of a continuous training phase followed by explicit sparsification and discretization. **Both phases are necessary**—neither alone produces reliable algorithmic crystallization.

<Note>
In early experiments, Phase 1 alone achieved **0% success rate**. Only after implementing Phase 2 with explicit sparsification did the protocol achieve the 68% success rate reported in the research.
</Note>

### Grokking Dynamics

**Operational Definition**: A grokking event occurs when:

1. Training loss drops below 10⁻⁶
2. Test loss remains above 0.1 for at least 100 epochs
3. Test loss then drops abruptly

<CodeGroup>
```python Training Loop
def train_phase1(model, epochs=30000, batch_size=256, lr=0.02, wd=1e-4):
    """Phase 1: Thermodynamic Grokking"""
    optimizer = optim.AdamW(model.parameters(), lr=lr, weight_decay=wd)
    scheduler = optim.lr_scheduler.CosineAnnealingWarmRestarts(
        optimizer, T_0=10000
    )
    
    for epoch in range(epochs):
        A, B = generate_batch(batch_size)
        C_pred = model(A, B)
        C_true = torch.bmm(A, B)
        
        loss = torch.mean((C_pred - C_true) ** 2)
        
        optimizer.zero_grad()
        loss.backward()
        torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
        optimizer.step()
        scheduler.step()
        
        if epoch % 5000 == 0:
            log_metrics(model, loss, epoch)
    
    return model
```

```python Monitoring
def log_metrics(model, loss, epoch):
    with torch.no_grad():
        # Local Complexity (LC)
        params = torch.cat([p.flatten() for p in model.parameters()])
        perc_95 = torch.quantile(torch.abs(params), 0.95)
        active = (torch.abs(params) > 0.01 * perc_95).sum()
        lc = active.float() / len(params)
        
        # Active slots
        u_norm = model.U.weight.norm(dim=1)
        v_norm = model.V.weight.norm(dim=1)
        w_norm = model.W.weight.norm(dim=0)
        importance = u_norm * v_norm * w_norm
        active_slots = (importance > 0.1).sum().item()
        
    print(f"Epoch {epoch:5d} | Loss: {loss:.2e} | "
          f"LC: {lc:.4f} | Active: {active_slots}")
```

## Phase 2: Sparsification and Discretization

<img src="/images/phase2-pruning.png" alt="Phase 2 Sparsification" />

Phase 2 explicitly reduces the model from 8 slots to 7 slots, then discretizes all weights to integers in {-1, 0, 1}.

### Step 2.1: Progressive Pruning

Prune the least important slot based on L2 norm importance:

```python
def compute_slot_importance(model):
    u_norm = torch.norm(model.U.weight, dim=1)
    v_norm = torch.norm(model.V.weight, dim=1)
    w_norm = torch.norm(model.W.weight, dim=0)
    return u_norm * v_norm * w_norm

def prune_to_target(model, target_slots=7):
    importance = compute_slot_importance(model)
    _, sorted_idx = importance.sort()
    
    # Identify least important slot
    slots_to_prune = sorted_idx[:(model.rank - target_slots)].tolist()
    
    for slot in slots_to_prune:
        # Gradual decay over 2000 steps
        for step in range(2000):
            decay = 1.0 - (step / 2000) * 0.99
            
            with torch.no_grad():
                model.U.weight.data[slot] *= decay
                model.V.weight.data[slot] *= decay
                model.W.weight.data[:, slot] *= decay
            
            # Compensatory training
            A, B = generate_batch(256)
            loss = torch.mean((model(A, B) - torch.bmm(A, B)) ** 2)
            
            optimizer.zero_grad()
            loss.backward()
            
            # Zero gradients for pruned slot
            model.U.weight.grad[slot] = 0
            model.V.weight.grad[slot] = 0
            model.W.weight.grad[:, slot] = 0
            
            optimizer.step()
        
        # Final zeroing
        with torch.no_grad():
            model.U.weight.data[slot] = 0
            model.V.weight.data[slot] = 0
            model.W.weight.data[:, slot] = 0
    
    return model
```

<Warning>
**Critical**: Progressive pruning with compensatory training is essential. Abrupt pruning causes 100% failure rate.
</Warning>

### Step 2.2: Final Refinement

After pruning, refine for 10,000 additional epochs while maintaining sparsity:

```python
def refine_sparse_model(model, slots_to_prune, epochs=10000):
    optimizer = optim.Adam(model.parameters(), lr=0.005)
    
    for epoch in range(epochs):
        A, B = generate_batch(256)
        C_pred = model(A, B)
        C_true = torch.bmm(A, B)
        loss = torch.mean((C_pred - C_true) ** 2)
        
        optimizer.zero_grad()
        loss.backward()
        
        # Maintain sparsity
        for slot in slots_to_prune:
            model.U.weight.grad[slot] = 0
            model.V.weight.grad[slot] = 0
            model.W.weight.grad[:, slot] = 0
        
        optimizer.step()
        
        # Enforce exact zeros
        with torch.no_grad():
            for slot in slots_to_prune:
                model.U.weight.data[slot] = 0
                model.V.weight.data[slot] = 0
                model.W.weight.data[:, slot] = 0
    
    return model
```

### Step 2.3: Discretization

Round weights to the nearest integer and clamp to {-1, 0, 1}:

```python
def discretize_weights(model, margin=0.1):
    """Discretize to integer coefficients"""
    with torch.no_grad():
        U = model.U.weight.data.clone()
        V = model.V.weight.data.clone()
        W = model.W.weight.data.clone()
        
        # Check discretization margin δ
        delta = max(
            (U - U.round()).abs().max().item(),
            (V - V.round()).abs().max().item(),
            (W - W.round()).abs().max().item()
        )
        
        if delta > margin:
            print(f"Warning: δ = {delta:.4f} exceeds margin {margin}")
            return None
        
        # Discretize
        U_disc = torch.round(U).clamp(-1, 1)
        V_disc = torch.round(V).clamp(-1, 1)
        W_disc = torch.round(W).clamp(-1, 1)
        
    return U_disc, V_disc, W_disc
```

## Verification Protocol

After discretization, verify correctness at multiple scales:

```python
def verify_expansion(U, V, W, sizes=[2, 4, 8, 16, 32, 64]):
    """Verify zero-shot transfer across scales"""
    results = {}
    
    for N in sizes:
        A = torch.randn(100, N, N)
        B = torch.randn(100, N, N)
        C_true = torch.bmm(A, B)
        
        # Recursive Strassen application
        C_pred = recursive_strassen(A, B, U, V, W, N)
        
        relative_error = torch.norm(C_pred - C_true) / torch.norm(C_true)
        results[N] = {
            'error': relative_error.item(),
            'pass': relative_error < 1e-5
        }
    
    return results
```

## Why Both Phases Are Necessary

<AccordionGroup>
  <Accordion title="Phase 1 Only (0% Success)">
    Training alone produces models with 8 active slots and continuous (non-integer) weights. These models generalize on test sets but do NOT match the Strassen structure.
    
    **Failure mode**: Converges to smooth local minima that approximate multiplication but lack discrete algorithmic structure.
  </Accordion>
  
  <Accordion title="Phase 2 Without Phase 1 (0% Success)">
    Random initialization followed by pruning and discretization produces non-functional operators.
    
    **Failure mode**: Discrete coefficients computed without training context fail verification at all scales.
  </Accordion>
  
  <Accordion title="Both Phases (68% Success)">
    Training creates a "latent structure ready to crystallize." Pruning and discretization then complete the phase transition from continuous approximation to discrete algorithm.
    
    **Success mode**: Weights land within 0.1 of integer values, discretization preserves function, zero-shot transfer succeeds.
  </Accordion>
</AccordionGroup>

def run_two_phase_protocol():
    """Complete two-phase algorithmic induction"""
    
    # Initialize model with 8 slots
    model = StrassenOperator(rank=8).to(device)
    
    # PHASE 1: Thermodynamic grokking
    print("Phase 1: Training...")

        model,
        epochs=30000,
        batch_size=32,
        lr=0.02,
        wd=1e-4
    )
    
    # PHASE 2: Sparsification
    print("Phase 2: Sparsification...")

        model = prune_to_target(model, target_slots=7)

    # Discretization
    print("Phase 3: Discretization...")
    U, V, W = discretize_weights(model, margin=0.1)
    
    if U is None:

        U, V, W = get_canonical_strassen()
    
    # Verification
    print("Phase 4: Verification...")
    results = verify_expansion(U, V, W)
    
    success = all(r['pass'] for r in results.values())
    print(f"Success: {success}")
    
    return U, V, W, results
