## Overview

This research demonstrates a reproducible two-phase protocol for inducing the Strassen matrix multiplication algorithm in neural networks with a 68% success rate. The work bridges materials science, statistical thermodynamics, and deep learning to understand how discrete algorithmic structures emerge during training.

## Quick Start

<Steps>
  <Step title="Installation">
    Install the package and dependencies
    
    ```bash
    git clone https://github.com/grisuno/strass_strassen.git
    cd strass_strassen
    pip install -r requirements.txt
    ```
  </Step>
  
  <Step title="Run Training">
    Train a model with optimal conditions
    
    ```python
    from src.training.train_strassen import train_strassen
    
    # Train with optimal batch size range
    model = train_strassen(
        batch_size=32,
        weight_decay=1e-4,
        epochs=2000
    )
    ```
  </Step>
  
  <Step title="Verify Crystallization">
    Check if the algorithm crystallized
    
    ```python

    A = torch.randn(64, 64)
    B = torch.randn(64, 64)
    C = strassen(A, B)
    ```
  </Step>
</Steps>

## Research Contributions

This work makes several key contributions to understanding algorithmic learning in neural networks:

1. **Engineering Protocol**: Reproducible recipe for inducing algorithmic structure with documented success rates
2. **Predictive Metrics**: κ achieves perfect classification (AUC 1.000) between crystallized and non-crystallized states
3. **Basin Characterization**: Discrete solutions remain stable under 50% pruning, establishing structural integrity
4. **Verification Framework**: Explicit criteria distinguishing genuine algorithmic learning from local minima
5. **Thermodynamic Foundation**: Measurable temperature, entropy, and heat capacity during training
6. **Statistical Validation**: 245 runs confirming batch size significantly affects crystallization
