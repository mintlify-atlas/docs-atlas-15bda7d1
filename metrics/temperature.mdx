## Overview

**T_eff** (effective temperature) is a thermodynamic metric derived from gradient noise that quantifies the "thermal energy" of the training dynamics. It achieves 100% phase classification accuracy, separating crystalline states (T_eff ≈ 10⁻¹⁷) from glass states (T_eff ≈ 10⁻¹⁶ to 10⁻⁵).

<Note>
  Crystalline states give T_eff ≈ 1×10⁻¹⁷. Glass states scatter between 1×10⁻¹⁶ and 8×10⁻⁵. The lowest glass temperature remains **an order of magnitude above the crystal ceiling**.
</Note>

## Mathematical Definition

T_eff is computed from the gradient covariance matrix via the fluctuation-dissipation relation:

```python
T_eff = (1/d) * Tr(Σ)

where:
  Σ = gradient covariance at final epoch
  d = number of parameters
  Tr(·) = matrix trace
```

### Thermodynamic Interpretation

<ParamField path="gradient_covariance" type="matrix" required>
  **Σ_t = Cov(g_t | θ_t)**
  
  Covariance of gradient noise at parameters θ_t
  
  ```python
  g_t = ∇_θ L(θ_t; B_t)  # Gradient at step t
  Σ_t = Cov(g_t)          # Across mini-batches
  ```
</ParamField>

<ParamField path="effective_temperature" type="float" required>
  **T_eff = (1/d) Tr(Σ)**
  
  Average variance per parameter dimension
  
  Analogous to: E = k_B T in statistical mechanics
</ParamField>

## Computing T_eff

<CodeGroup>

```python Basic Implementation
import torch

def compute_effective_temperature(model, data_loader, device='cuda'):
    """Compute T_eff from gradient covariance."""
    model.eval()
    gradients = []
    
    # Collect gradients over multiple batches
    for batch_idx, (x, y) in enumerate(data_loader):
        if batch_idx >= 20:  # Sample 20 batches
            break
        
        x, y = x.to(device), y.to(device)
        model.zero_grad()
        
        # Forward pass
        output = model(x)
        loss = torch.nn.functional.mse_loss(output, y)
        
        # Backward pass
        loss.backward()
        
        # Collect gradient vector
        grad = torch.cat([
            p.grad.flatten() 
            for p in model.parameters() 
            if p.grad is not None
        ])
        gradients.append(grad.cpu())
    
    # Stack gradients
    gradients = torch.stack(gradients)  # [n_batches, n_params]
    
    # Compute covariance
    Sigma = torch.cov(gradients.T)  # [n_params, n_params]
    
    # Compute temperature
    d = Sigma.shape[0]
    T_eff = torch.trace(Sigma).item() / d
    
    return T_eff

# Usage
T_eff = compute_effective_temperature(model, train_loader)
print(f"T_eff = {T_eff:.2e}")

if T_eff < 1e-16:
    print("[OK] CRYSTAL phase detected")
else:
    print("[INFO] GLASS phase")
```

```python Memory-Efficient Version
def compute_temperature_efficient(model, data_loader, n_samples=20):
    """Memory-efficient T_eff computation."""
    model.eval()
    
    # Accumulate first and second moments
    n_params = sum(p.numel() for p in model.parameters())
    mean_grad = torch.zeros(n_params)
    sum_squared = torch.zeros(n_params)
    
    for batch_idx, (x, y) in enumerate(data_loader):
        if batch_idx >= n_samples:
            break
        
        model.zero_grad()
        output = model(x)
        loss = torch.nn.functional.mse_loss(output, y)
        loss.backward()
        
        grad = torch.cat([
            p.grad.flatten() for p in model.parameters() if p.grad is not None
        ]).cpu()
        
        mean_grad += grad
        sum_squared += grad ** 2
    
    # Compute variance (diagonal approximation)
    mean_grad /= n_samples
    variance = (sum_squared / n_samples) - mean_grad ** 2
    
    # Temperature = mean variance
    T_eff = variance.mean().item()
    
    return T_eff
```

```python From Checkpoint
def temperature_from_checkpoint(checkpoint_path):
    """Estimate T_eff from saved checkpoint."""
    checkpoint = torch.load(checkpoint_path)
    
    # Option 1: If gradients are saved
    if 'gradient_covariance' in checkpoint:
        Sigma = checkpoint['gradient_covariance']
        d = Sigma.shape[0]
        return torch.trace(Sigma).item() / d
    
    # Option 2: Estimate from weight distribution
    coeffs = checkpoint['coeffs']
    flat_params = torch.cat([c.flatten() for c in coeffs.values()])
    
    # Temperature proxy: variance of weights
    T_proxy = torch.var(flat_params).item()
    
    return T_proxy
```

## Phase Classification

T_eff provides perfect separation between phases:

| Phase | T_eff Range | Interpretation |
|-------|-------------|----------------|
| **Crystal** | ≈ 10⁻¹⁷ | Ground state, discrete weights |
| **Glass** | 10⁻¹⁶ to 10⁻⁵ | Metastable, continuous weights |
| **Transition** | ~10⁻¹⁶ | Phase boundary (rare) |

<Info>
  **The temperature gap is at least one order of magnitude**, enabling 100% classification accuracy in the tested dataset.
</Info>

### Classification Algorithm

        'description': description,

## Theoretical Foundation

### Fluctuation-Dissipation Relation

T_eff emerges from the fluctuation-dissipation theorem:

<ParamField path="fluctuation_dissipation" type="theory">
  In equilibrium: ⟨ΔE²⟩ = k_B T × (heat capacity)
  
  For neural networks:
  - Energy ↔ Loss L(θ)
  - Fluctuations ↔ Gradient variance Σ
  - Temperature ↔ T_eff = Tr(Σ) / d
</ParamField>

### Connection to Statistical Mechanics

```python
# Classical partition function
Z = ∫ exp(-E(θ) / T) dθ

# In discrete regime (T → 0):
Z → sum over discrete states

# Crystal state:
T_eff ≈ 0 → Z dominated by single discrete configuration

# Glass state:
T_eff >> 0 → Z has many competing configurations
```

## Entropy and Heat Capacity

T_eff relates to thermodynamic entropy:

```python
def compute_entropy_from_temperature(T_eff, n_params):
    """Estimate entropy from temperature."""
    if T_eff < 1e-20:
        return 0.0  # Crystal: S = 0
    
    # Equipartition: S ∝ d * ln(T_eff)
    S = n_params * np.log(T_eff)
    return S

# Example
S_crystal = compute_entropy_from_temperature(T_eff=1e-17, n_params=21)
S_glass = compute_entropy_from_temperature(T_eff=1e-5, n_params=21)

print(f"Crystal entropy: {S_crystal:.2f}")
print(f"Glass entropy: {S_glass:.2f}")
print(f"Entropy difference: {S_glass - S_crystal:.2f}")
```

<Info>

## Practical Example: Complete Thermodynamic Analysis

```python
import torch
import numpy as np
from typing import Dict, List, Tuple

class ThermodynamicAnalyzer:
    """Complete thermodynamic analysis of training."""
    
    def __init__(self, model, n_params):
        self.model = model
        self.n_params = n_params
        self.history = []
    
    def measure(self, epoch, data_loader):
        """Measure all thermodynamic quantities."""
        # Temperature
        T_eff = compute_temperature_efficient(
            self.model, data_loader, n_samples=10
        )
        
        # Other metrics
        coeffs = self.model.get_coefficients()
        delta = compute_delta(coeffs)
        kappa = compute_kappa_quantum(coeffs)
        lc = compute_local_complexity(coeffs)
        
        # Derived quantities
        S = self.n_params * np.log(T_eff) if T_eff > 1e-20 else 0.0
        phase = "crystal" if T_eff < 1e-16 else "glass"
        
        record = {
            'epoch': epoch,
            'T_eff': T_eff,
            'entropy': S,
            'delta': delta,
            'kappa': kappa,
            'lc': lc,
            'phase': phase
        }
        
        self.history.append(record)
        return record
    
    def get_phase_transition_epoch(self):
        """Find epoch where phase transition occurred."""
        for i in range(1, len(self.history)):
            prev = self.history[i-1]
            curr = self.history[i]
            
            if prev['phase'] == 'glass' and curr['phase'] == 'crystal':
                return curr['epoch']
        
        return None
    
    def compute_heat_capacity(self, window=10):
        """Estimate heat capacity from temperature fluctuations."""
        if len(self.history) < window:
            return None
        
        # Recent temperature variance
        recent_temps = [
            rec['T_eff'] for rec in self.history[-window:]
        ]
        
        # C_V ∝ ⟨(ΔE)²⟩ / T²
        temp_var = np.var(recent_temps)
        mean_temp = np.mean(recent_temps)
        
        if mean_temp < 1e-20:
            return 0.0
        
        C_V = temp_var / (mean_temp ** 2)
        return C_V
    
    def summarize(self):
        """Print thermodynamic summary."""
        if not self.history:
            print("No measurements recorded.")
            return
        
        initial = self.history[0]
        final = self.history[-1]
        transition_epoch = self.get_phase_transition_epoch()
        heat_capacity = self.compute_heat_capacity()
        
        print("\n" + "="*60)
        print("THERMODYNAMIC SUMMARY")
        print("="*60)
        
        print(f"\nInitial State (epoch {initial['epoch']}):")
        print(f"  T_eff = {initial['T_eff']:.2e}")
        print(f"  S = {initial['entropy']:.2f}")
        print(f"  Phase: {initial['phase']}")
        
        print(f"\nFinal State (epoch {final['epoch']}):")
        print(f"  T_eff = {final['T_eff']:.2e}")
        print(f"  S = {final['entropy']:.2f}")
        print(f"  Phase: {final['phase']}")
        
        if transition_epoch:
            print(f"\nPhase Transition:")
            print(f"  Occurred at epoch {transition_epoch}")
            print(f"  ΔT = {initial['T_eff'] - final['T_eff']:.2e}")
            print(f"  ΔS = {initial['entropy'] - final['entropy']:.2f}")
        
        if heat_capacity is not None:
            print(f"\nHeat Capacity: C_V = {heat_capacity:.2e}")

# Usage
analyzer = ThermodynamicAnalyzer(model, n_params=21)

for epoch in range(3000):
    train_epoch(model, optimizer, train_loader)
    
    if epoch % 100 == 0:
        record = analyzer.measure(epoch, train_loader)
        print(f"Epoch {epoch}: T={record['T_eff']:.2e}, phase={record['phase']}")

analyzer.summarize()
```
