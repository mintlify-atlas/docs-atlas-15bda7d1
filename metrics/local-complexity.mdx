## Overview

**Local Complexity (LC)** measures the fraction of active parameters in a network, capturing the transition from high-dimensional optimization to low-dimensional algorithmic structure. LC falls to zero exactly at the grokking transition, confirming it as a **first-order phase change indicator**.

<Note>
  Local complexity falls from 442 to ~0 at epoch 2160, coinciding exactly with the grokking transition (Figure 6 in the paper).
</Note>

## Mathematical Definition

LC quantifies what fraction of parameters are "active" (contributing significantly to the computation):

```python
LC(θ) = (n_active / n_total)

where:
  n_active = count(|θ_i| > threshold)
  threshold = 0.01 × quantile_95(|θ|)
```

### Formal Definition

<ParamField path="active_parameters" type="count" required>
  Parameters whose absolute value exceeds 1% of the 95th percentile
  
  ```python
  perc_95 = torch.quantile(torch.abs(params), 0.95)
  active = (torch.abs(params) > 0.01 * perc_95).sum()
  ```
</ParamField>

<ParamField path="local_complexity" type="float" range="[0, 1]">
  **LC = active_count / total_params**
  
  - LC ≈ 1.0: All parameters active (early training)
  - LC → 0: Only discrete structure remains (crystallization)
</ParamField>

## Computing Local Complexity

<CodeGroup>

```python Basic Implementation
import torch

def compute_local_complexity(coeffs):
    """Compute local complexity from model coefficients."""
    # Flatten all parameters
    flat_params = torch.cat([
        c.flatten() for c in coeffs.values()
    ])
    
    # Compute 95th percentile
    perc_95 = torch.quantile(torch.abs(flat_params), 0.95)
    
    # Count active parameters (above 1% of 95th percentile)
    threshold = 0.01 * perc_95
    active = (torch.abs(flat_params) > threshold).sum()
    
    # Return fraction
    lc = (active.float() / len(flat_params)).item()
    
    return lc

# Example usage
model_coeffs = model.get_coefficients()
lc = compute_local_complexity(model_coeffs)

print(f"Local Complexity: {lc:.4f}")
if lc < 0.01:
    print("[OK] Crystallized - only discrete structure remains")
elif lc < 0.5:
    print("[INFO] Sparsifying - approaching crystallization")
else:
    print("[INFO] Dense - early training phase")
```

```python With Threshold Analysis
def analyze_complexity_distribution(coeffs, thresholds=[0.001, 0.01, 0.1]):
    """Analyze LC at multiple threshold levels."""
    flat_params = torch.cat([c.flatten() for c in coeffs.values()])
    perc_95 = torch.quantile(torch.abs(flat_params), 0.95)
    
    results = {}
    for thresh_mult in thresholds:
        threshold = thresh_mult * perc_95
        active = (torch.abs(flat_params) > threshold).sum()
        lc = (active.float() / len(flat_params)).item()
        results[thresh_mult] = lc
    
    return results

# Example
results = analyze_complexity_distribution(model.get_coefficients())
for thresh, lc in results.items():
    print(f"Threshold {thresh:.3f}×p95: LC = {lc:.4f}")
```

## Grokking Transition Detection

LC's most important property is its behavior at the grokking transition:

### Empirical Observation

```python
# From Figure 6 in the paper
Epoch 2000: LC = 442  # High-dimensional search
Epoch 2160: LC = ~0   # Sudden collapse
Epoch 2500: LC = 0    # Stable crystallization

# Training loss < 1e-6 throughout
# Test loss drops from 0.5 → 1e-6 at epoch 2160
```

<Info>
  **LC falling to zero at epoch 2160 confirms that it captures the phase change.** This coincides exactly with the grokking transition where test loss suddenly drops.
</Info>

### Phase Classification by LC

| LC Value | Phase | Interpretation |
|----------|-------|----------------|
| > 0.8 | Early Training | All parameters contributing |
| 0.5 - 0.8 | Convergence | Loss decreasing, still dense |
| 0.1 - 0.5 | Pre-grokking | Sparsification beginning |
| < 0.1 | Grokking | Algorithmic structure emerging |
| ≈ 0 | Crystal | Only discrete coefficients remain |

## Monitoring LC During Training

<Steps>

  </Step>
  
  <Step title="Compute Periodically">
    Track LC evolution

    ```python
    for epoch in range(epochs):

        if epoch % 10 == 0:
            coeffs = model.get_coefficients()
            lc = compute_local_complexity(coeffs)
            lc_history.append((epoch, lc))
    ```
  </Step>
  
  <Step title="Detect Grokking">
    Watch for sudden LC collapse
    
    ```python
            # Check for grokking transition
            if len(lc_history) > 5:
    ```

## Relationship to Training Dynamics

### LC vs Loss Curves

LC provides information orthogonal to loss:

```python
# Typical grokking trajectory

# Phase 1: Early training (epochs 0-500)
train_loss: 1.0 → 1e-6   # Rapid decrease
test_loss:  1.0 → 0.5    # Slight decrease
LC:         1.0 → 0.8    # Mild sparsification

# Phase 2: Grokking plateau (epochs 500-2160)
train_loss: < 1e-6       # Converged
test_loss:  0.5          # Stuck
LC:         0.8 → 0.5    # Slow sparsification

# Phase 3: Grokking (epoch 2160)
train_loss: < 1e-6       # Still converged
test_loss:  0.5 → 1e-6   # SUDDEN DROP
LC:         0.5 → 0      # COLLAPSE TO ZERO

# Phase 4: Crystallization (epochs 2160+)
train_loss: < 1e-6
test_loss:  < 1e-6
LC:         ≈ 0          # Stable
```

<Warning>
  **Loss curves alone cannot detect grokking before it happens.** LC provides early warning by showing sparsification dynamics.
</Warning>

## Theoretical Interpretation

### Effective Dimension

LC measures the effective dimensionality of the solution space:

```python
# Effective number of parameters
n_eff = LC × n_total

# Example:
n_total = 84    # 7×4 + 7×4 + 4×7 for Strassen
LC = 0.0        # After grokking
n_eff = 0       # Zero-dimensional attractor

# Before grokking:
LC = 0.5
n_eff = 42      # Still 42 effective dimensions
```

### Connection to Phase Transitions

LC as order parameter:

<ParamField path="order_parameter" type="metric">
  **Φ_LC = 1 - LC**
  
  - Φ_LC = 0: Disordered phase (all parameters active)
  - Φ_LC = 1: Ordered phase (only discrete structure)
  
  First-order transition: Φ_LC jumps discontinuously from 0 → 1
</ParamField>

## Practical Example: Real-Time Monitoring

Complete training loop with LC monitoring:

```python
import torch
import torch.nn as nn
from typing import List, Tuple

class GrokkingMonitor:
    """Monitor grokking via Local Complexity."""
    
    def __init__(self, model, check_interval=10):
        self.model = model
        self.check_interval = check_interval
        self.lc_history: List[Tuple[int, float]] = []
        self.grokking_epoch = None
    
    def step(self, epoch):
        """Check LC at current epoch."""
        if epoch % self.check_interval != 0:
            return None
        
        coeffs = self.model.get_coefficients()
        lc = compute_local_complexity(coeffs)
        self.lc_history.append((epoch, lc))
        
        # Detect grokking
        if self.grokking_epoch is None and lc < 0.01:
            if len(self.lc_history) > 1:
                prev_lc = self.lc_history[-2][1]
                if prev_lc > 0.5:
                    self.grokking_epoch = epoch
                    return {
                        'event': 'grokking_detected',
                        'epoch': epoch,
                        'lc_before': prev_lc,
                        'lc_after': lc
                    }
        
        return {'lc': lc}
    
    def get_summary(self):
        """Summarize grokking dynamics."""
        if not self.lc_history:
            return {}
        
        epochs, lcs = zip(*self.lc_history)
        return {
            'initial_lc': lcs[0],
            'final_lc': lcs[-1],
            'grokking_epoch': self.grokking_epoch,
            'lc_drop': lcs[0] - lcs[-1]
        }

# Usage
model = BilinearStrassenModel()
monitor = GrokkingMonitor(model, check_interval=10)

for epoch in range(3000):
    train_epoch(model, optimizer, train_loader)
    
    event = monitor.step(epoch)
    if event and 'event' in event:
        print(f"\n🔔 {event['event']} at epoch {event['epoch']}")
        print(f"   LC: {event['lc_before']:.3f} → {event['lc_after']:.4f}")

# Final summary
summary = monitor.get_summary()
print(f"\nTraining Summary:")
print(f"  Initial LC: {summary['initial_lc']:.3f}")
print(f"  Final LC: {summary['final_lc']:.4f}")
if summary['grokking_epoch']:
    print(f"  Grokking at: epoch {summary['grokking_epoch']}")
```
