## Overview

**δ** (delta) is the discretization margin that measures the L-infinity distance between continuous weights and their nearest discrete values. It serves as both a **success criterion** for algorithmic crystallization and an **early prospecting tool** for seed selection.

<Note>
  δ = 0 exactly at the crystal phase; δ ≈ 0.49 for glass states. The transition is abrupt with no intermediate checkpoints.
</Note>

## Mathematical Definition

δ measures the maximum distance from any weight to its rounded integer value:

```python
δ(θ) = ||θ - Q(θ)||_∞

where:
  Q(θ) = round_and_clamp(θ, {-1, 0, 1})
  ||·||_∞ = max absolute value
```

### Formal Definition

<ParamField path="discretization_operator" type="function" required>
  **Q(θ):** Post-hoc projection of coefficients to discrete grid
  
  In this work: round and clamp to {-1, 0, 1}
</ParamField>

<ParamField path="discretization_margin" type="float" required>
  **δ(θ) = ||θ - Q(θ)||_∞**
  
  A solution is "discretizable" if δ(θ) ≤ 0.1
</ParamField>

## Computing δ

Simple and efficient computation:

<CodeGroup>

```python Basic Implementation
import torch

def compute_delta(coeffs):
    """Compute discretization margin δ."""
    deltas = []
    for name, tensor in coeffs.items():
        # Distance to nearest integer
        error = (tensor - tensor.round()).abs().max().item()
        deltas.append(error)
    
    return max(deltas)

# Example usage
model_coeffs = {
    'U': model.U.weight.data,
    'V': model.V.weight.data,
    'W': model.W.weight.data
}

delta = compute_delta(model_coeffs)
print(f"δ = {delta:.4f}")

if delta < 0.1:
    print("[OK] Discretizable - within rounding margin")
else:
    print("[FAIL] Not discretizable")
```

## Phase Transition Behavior

δ exhibits a sharp transition at algorithmic crystallization:

| Phase | δ Value | Interpretation |
|-------|---------|----------------|
| Crystal | 0.000 | Exact discrete coefficients in {-1, 0, 1} |
| Near-Crystal | 0.001 - 0.1 | Within rounding margin |
| Polycrystal | 0.1 - 0.3 | Partial structure |
| Glass | > 0.3 | No discrete structure, typical ≈ 0.49 |

<Info>
  **The transition from glass (δ ≈ 0.49) to crystal (δ = 0) is a first-order phase transition with no intermediate states.**
</Info>

## Early Seed Prospecting

δ's most powerful application is **seed mining** - selecting promising initialization seeds before full training:

<Steps>
  <Step title="Initialize Multiple Seeds">
    Train N candidate seeds for a few epochs
    
    ```python
    seeds = [42, 100, 200, 300, 400]
    early_deltas = {}
    
    for seed in seeds:
        set_seed(seed)
        model = BilinearStrassenModel()
        
        # Train for 50-100 epochs only
        train_for_epochs(model, n_epochs=100)
        
        # Measure initial entropy via δ
        delta = compute_delta(model.get_coefficients())
        early_deltas[seed] = delta
    ```
  </Step>
  
  <Step title="Rank by δ">
    Select seeds with lowest early δ
    
    ```python
    # Sort seeds by early delta
    ranked_seeds = sorted(
        early_deltas.items(), 
        key=lambda x: x[1]
    )
    
    print("Seed Ranking (by δ after 100 epochs):")
    for seed, delta in ranked_seeds[:5]:
        print(f"  Seed {seed}: δ = {delta:.4f}")
    
    # Select best seed
    best_seed = ranked_seeds[0][0]
    print(f"\nSelected seed: {best_seed}")
    ```
  </Step>
  
  <Step title="Full Training">
    Train only the most promising seeds
    
    ```python
    # Train top 3 seeds to completion
    for seed, early_delta in ranked_seeds[:3]:
        set_seed(seed)
        model = BilinearStrassenModel()
        
        print(f"Training seed {seed} (early δ={early_delta:.4f})...")
        train_full(model, epochs=3000)
        
        final_delta = compute_delta(model.get_coefficients())
        print(f"  Final δ = {final_delta:.4f}")
    ```
  </Step>
</Steps>

<Warning>
  **δ transforms cherry-picking into seed mining.** Measure initial entropy across seeds to identify which ones are heading toward the algorithmic attractor.
</Warning>

## Success Criterion

δ defines the formal success criterion for discretization:

<ParamField path="discretization_success" type="boolean">
  A run achieves discretization success if and only if:
  
  - All 21 weight values satisfy: `|w - round(w)| < 0.5`
  - AND the rounded values match valid Strassen coefficients
  - Threshold: **δ ≤ 0.1**
</ParamField>

```python
def check_discretization_success(model, threshold=0.1):
    """Formal verification of discretization success."""
    coeffs = model.get_coefficients()
    delta = compute_delta(coeffs)
    
    # Condition 1: Within rounding margin
    if delta > threshold:
        return False, f"δ={delta:.4f} exceeds threshold {threshold}"
    
    # Condition 2: Structural match
    Q_coeffs = {k: v.round() for k, v in coeffs.items()}
    if not verify_strassen_structure(Q_coeffs):
        return False, "Rounded coefficients don't match Strassen"
    
    return True, f"Success: δ={delta:.4f}"

success, message = check_discretization_success(model)
print(message)
```

def compute_alpha_purity(delta):
    """Alpha purity: α = -log(δ)."""

        return 20.0  # α_max for perfect discretization
    return -np.log(delta)

## Practical Workflow

Complete workflow using δ for seed selection and verification:

```python
import numpy as np
import torch

class SeedProspector:
    """Early seed selection using δ."""
    
    def __init__(self, n_seeds=10, early_epochs=100):
        self.n_seeds = n_seeds
        self.early_epochs = early_epochs
        self.results = {}
    
    def prospect(self, train_fn):
        """Evaluate multiple seeds and rank them."""
        seeds = np.random.randint(0, 10000, self.n_seeds)
        
        for seed in seeds:
            set_seed(seed)
            model = BilinearStrassenModel()
            
            # Short training
            train_fn(model, epochs=self.early_epochs)
            
            # Measure early δ
            coeffs = model.get_coefficients()
            delta = compute_delta(coeffs)
            
            self.results[seed] = {
                'early_delta': delta,
                'model_state': model.state_dict().copy()
            }
        
        # Rank by delta
        self.ranked = sorted(
            self.results.items(),
            key=lambda x: x[1]['early_delta']
        )
        
        return self.ranked
    
    def get_best_seeds(self, n=3):
        """Return top n seeds by early δ."""
        return [seed for seed, _ in self.ranked[:n]]

# Usage
prospector = SeedProspector(n_seeds=20, early_epochs=100)
ranked_seeds = prospector.prospect(train_function)

print("Top 5 seeds:")
for seed, data in ranked_seeds[:5]:
    print(f"  Seed {seed}: early δ = {data['early_delta']:.4f}")

# Train only best seeds
best_seeds = prospector.get_best_seeds(n=3)
for seed in best_seeds:
    print(f"\nFull training on seed {seed}...")
    # Train to completion
```
