# Experimental Results

This page presents the complete experimental results from the Riemann-Strassen crystallization research, documenting 245 independent training runs that demonstrate successful algorithmic structure induction in neural networks.

## Overview

The experimental program investigated whether neural networks can be engineered to learn the exact Strassen matrix multiplication algorithm rather than approximating it. Results show a **68% success rate** (133/195 runs) in achieving crystallization to the canonical Strassen structure.

<Note>
The optimal batch size range [24-128] creates a "Goldilocks zone" where:
- **Small batches (8-16):** Too unstable, high gradient noise
- **Optimal batches (24-128):** Ready to crystallize, moderate entropy
- **Large batches (256-512):** Trapped in glass, high h̄ entropy
</Note>

## Failure Mode Analysis

### 50-Run Dedicated Failure Study

**Configuration:** Batch size 32, 50 different random seeds

**Results:**
- Success: 26/50 (52%)
- Consistent with expected variation
- All failures showed κ > 1000
- Failed runs converged to local minima with δ ≈ 0.49

### Common Failure Patterns

<AccordionGroup>
  <Accordion title="Type 1: Premature Convergence">
    - Loss plateaus before reaching crystallization basin
    - κ remains > 100
    - δ stabilizes around 0.45-0.50
    - Training accuracy: 100%, but weights not near integers
  </Accordion>
  
  <Accordion title="Type 2: Phase Transition Failure">
    - Initial grokking observed (acc: 0% → 100%)
    - κ drops to ~10 but fails to reach 1.0
    - System trapped in "warm glass" state
    - Requires restart with different seed
  </Accordion>
  
  <Accordion title="Type 3: Symmetry Breaking Failure">
    - All 8 slots remain active (should prune to 7)
    - Weights don't align to discrete grid
    - Expansion operator fails verification
    - Fallback to canonical coefficients required
  </Accordion>
</AccordionGroup>

## Reproducibility

### Exact Reproduction Protocol

To reproduce the 68% success rate:

1. **Environment:**
   - Python 3.8+
   - PyTorch 2.0+
   - CPU or CUDA (results consistent)

2. **Hyperparameters:**
   ```python
   batch_size = 32  # or any in [24, 128]
   weight_decay = 1e-4
   learning_rate = 1e-3
   epochs = 10000  # early stopping at crystallization
   optimizer = "AdamW"
   ```

3. **Training Conditions:**
   - Monitor κ every 100 epochs
   - Stop when κ < 1.1 and δ < 0.1
   - Typical training time: 15-45 minutes (CPU)

4. **Verification:**
   - Prune to 7 slots (L2 importance)
   - Round weights to {-1, 0, 1}
   - Test on sizes [2, 4, 8, 16, 32, 64]
