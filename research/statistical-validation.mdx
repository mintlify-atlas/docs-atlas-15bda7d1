# Statistical Validation

This page presents the complete statistical validation of the Strassen algorithm crystallization phenomenon, including ANOVA results, effect size analysis, predictive metric validation, and hypothesis testing.

## Overview

Statistical analysis of 245 training runs confirms that:

1. **Batch size significantly affects crystallization** (F=15.34, p less than 0.0001, η²=0.244)
2. **κ achieves perfect prediction** (AUC = 1.000, 95% CI [1.000, 1.000])
3. **First-order phase transition observed** (no intermediate states)
4. **Results are reproducible** (bootstrap validation, n=10,000)

## ANOVA Results

### Primary Analysis (N = 195)

Two-way ANOVA examining effects of batch size and protocol on discretization quality:

```text
================================================================================
ANOVA Table: Discretization Error (δ) by Batch Size and Protocol
================================================================================

Source          | SS      | df  | MS      | F      | p-value    | η²    |
----------------|---------|-----|---------|--------|------------|-------|
Batch Size      | 0.287   | 4   | 0.072   | 15.34  | < 0.0001** | 0.244 |
Protocol        | 0.052   | 1   | 0.052   | 11.08  | 0.001**    | 0.044 |
Interaction     | 0.014   | 4   | 0.004   | 0.75   | 0.558      | 0.012 |
Error           | 0.883   | 189 | 0.005   | -      | -          | -     |
----------------|---------|-----|---------|--------|------------|-------|
Total           | 1.236   | 198 | -       | -      | -          | 1.000 |

** p < 0.01 (highly significant)
η² = Sum of Squares (effect) / Sum of Squares (total)
```

### Interpretation

<Accordion title="Batch Size Effect">
  **F-statistic:** 15.34  
  **p-value:** < 0.0001  
  **Effect size (η²):** 0.244
  
  Batch size explains **24.4% of the variance** in discretization quality. This is considered a **large effect** in experimental research (Cohen's guidelines: small=0.01, medium=0.06, large=0.14).
  
  The extremely low p-value (< 0.0001) indicates that this effect is highly unlikely to occur by chance (probability < 0.01%).
</Accordion>

<Accordion title="Protocol Effect">
  **F-statistic:** 11.08  
  **p-value:** 0.001  
  **Effect size (η²):** 0.044
  
  The choice of protocol (A vs B) accounts for **4.4% of variance**. While statistically significant, this is a **small-to-medium effect**, suggesting both protocols perform similarly when batch size is controlled.
</Accordion>

<Accordion title="Interaction Effect">
  **F-statistic:** 0.75  
  **p-value:** 0.558  
  **Effect size (η²):** 0.012
  
  The interaction between batch size and protocol is **not significant** (p > 0.05). This indicates that the effect of batch size is consistent across both protocols—no evidence that one protocol is better for certain batch sizes.
</Accordion>
