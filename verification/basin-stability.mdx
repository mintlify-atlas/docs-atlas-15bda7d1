## Overview

Basin stability measures how robust the learned algorithmic structure is to perturbations. The discrete Strassen solution occupies a narrow region of weight space—but within this region, it exhibits surprising resilience to structured perturbations (pruning) while being extremely fragile to random noise.

This dual nature reveals fundamental properties about algorithmic crystallization in neural networks.

## Two Types of Perturbation

The research tests two fundamentally different perturbation types:

1. **Structured pruning**: Removing weights systematically by importance
2. **Random noise**: Adding Gaussian noise to all weights

These produce dramatically different outcomes, revealing that the basin is **narrow but deep**.

## Fragility to Random Noise

In stark contrast to pruning stability, the discrete solution is **extremely fragile** to random perturbations:

```python
def run_noise_stability_test(checkpoint_path, noise_levels=[0.001, 0.005, 0.01, 0.05, 0.1]):
    """Test stability under Gaussian noise perturbations"""
    model = load_checkpoint(checkpoint_path)
    
    for sigma in noise_levels:
        successes = 0
        n_trials = 100
        
        for trial in range(n_trials):
            model_copy = StrassenBilinear(rank=8)
            model_copy.U.data = model.U.data.clone() + torch.randn_like(model.U.data) * sigma
            model_copy.V.data = model.V.data.clone() + torch.randn_like(model.V.data) * sigma
            model_copy.W.data = model.W.data.clone() + torch.randn_like(model.W.data) * sigma
            
            S, _ = compute_S_theta(model_copy)
            successes += S
        
        rate = successes / n_trials
        print(f"  sigma={sigma:.3f}: {rate:.0%} success ({successes}/{n_trials})")
```

## Why This Matters

This extreme fragility is **not a limitation**—it's the fundamental justification for precise engineering of training conditions.

### Narrow Basin Confirmation

The noise sensitivity confirms that discrete algorithmic solutions occupy **narrow basins of attraction** in weight space. Even tiny perturbations (0.1% of typical weight magnitudes) push the system out of the region where rounding preserves correctness.

### Engineering Implications

Navigating toward stable algorithmic structure requires exact engineering control:

- Batch size must be in [24, 128]
- Weight decay ≥ 1e-4 is essential
- Training must continue beyond 1000 epochs
- Symmetric initialization helps

Without these precise conditions, the optimization trajectory won't reach the narrow discretization basin.

## The Polycrystalline Intermediate State

The pruning experiments reveal an intermediate state between crystal and glass:

| State | Description | Properties |
|-------|-------------|------------|
| **Crystal** | Fully formed Strassen structure | δ = 0, 7 active slots, κ = 1.000 |
| **Polycrystal** | Partially pruned but still functional | δ < 0.1, ~3.5 active slots (50% pruned), maintains accuracy |
| **Glass** | Failed discretization | δ ≈ 0.49, κ = 999999, cannot discretize |

The polycrystalline state demonstrates that the algorithmic structure has **genuine structural integrity**—it can lose half its weights and still function perfectly, as long as the remaining weights stay within the discrete basin.

## Phase Transition Nature

Both transitions exhibit characteristics of **first-order phase transitions**:

### Crystallization (Training)

```
Glass (κ=999999, δ≈0.49) → Crystal (κ=1.000, δ=0)
```

No intermediate states exist. The system either crystallizes or remains glass.

### Collapse (Pruning)

```
Crystal (50% sparse, accurate) → Collapse (51% sparse, failed)
```

Sharp transition with no gradual degradation.

## Implications for Deep Learning

The basin stability findings have broader implications:

### Reproducibility Crisis

<Warning>If a well-defined algorithm like Strassen requires such precise training conditions to emerge, what does this say about reproducibility in deep learning more broadly?</Warning>

Narrow basins containing algorithmic solutions may be much more common than believed. Our inability to reach them consistently may explain many reproducibility failures in the field.

### The Value of Fragility

Fragility transforms from an apparent weakness to a central insight: the extreme sensitivity confirms we're finding **true discrete attractors**, not approximate solutions. A robust solution might generalize well but not represent the exact algorithm.

### Delta as a Compass

The discretization margin δ serves as an early prospecting tool:

- Measure initial entropy of random seeds
- Run a few training epochs per seed
- Track which seeds have δ → 0
- Select seeds heading toward the algorithmic attractor

If κ is temperature, then **δ is our compass** for navigating weight space.
